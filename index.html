Требования

я вижу основной элемент с соответствующим id 'main-doc', который содержит основное содержание страницы (техническая документация).

В элементе # main-doc я вижу несколько элементов section, каждый со своим классом main-section. Там должно быть минимум 5.

первый элемент в каждом .main-разделе должен быть элементом заголовка, который содержит текст, описывающий тему этого раздела.

Каждый элемент section с классом main-section также должен иметь идентификатор, соответствующий тексту каждого заголовка, содержащегося в нем.
Любые пробелы должны быть заменены подчеркиванием (например, раздел, содержащий заголовок «Javascript и Java», должен иметь соответствующий id 'Javascript_and_Java').

элементы .main-section должны содержать не менее 10 элементов p (не каждый).

элементы .main-section должны содержать как минимум 5 элементов кода (не каждый).

элементы .main-section должны содержать как минимум 5 элементов li (не каждый).

я вижу элемент nav с соответствующим id 'navbar'.

элемент navbar должен содержать один элемент header, который содержит текст, описывающий тему технической документации. Кроме того, панель навигации должна содержать элементы link (a) с классом nav-link. Должен быть один для каждого элемента с классом main-section. элемент заголовка в панели навигации должен находиться перед любыми элементами ссылки (а) в панели навигации.

Каждый элемент с классом nav-link должен содержать текст, соответствующий тексту заголовка в каждом разделе (например, если у вас есть раздел /
заголовок «Hello world», ваша панель навигации должна иметь элемент, который содержит текст 'Привет, мир').

Когда я нажимаю на элемент navbar, страница должна перейти к соответствующему разделу элемента main-doc (например, если я нажимаю на элемент nav-link,
содержащий текст «Hello world», страница перемещается к элементу раздела, который имеет этот идентификатор и содержит соответствующий заголовок.

На устройствах обычного размера (ноутбуки, настольные компьютеры) элемент с id 'navbar' должен отображаться в левой части экрана и всегда должен быть видимым для пользователя. На моей странице технической документации должен использоваться хотя бы один медиазапрос.

    Это делается так:

  • #main-doc { display:flex; flex-direction: row; }
  • #navbar { display:flex; flex-direction: column; padding:20px; border: 4px solid grey; justify-content:space-between; }
  • .sectio{ margin-left:10px; width: 100%; padding: 40px; display:flex; flex-direction: column; border: 4px solid grey; justify-content:space-between; }
  • footer{ position: fixed; width: 100%; height:30px; text-align:center; background-color: darkslategrey; color:white; left:0; bottom: 0; }
  • a:visited {color: black;} @media screen { * {font-family: sans-serif;} } h3 {text-align: center;}
